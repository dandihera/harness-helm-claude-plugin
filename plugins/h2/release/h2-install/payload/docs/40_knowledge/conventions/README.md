# 40_knowledge/conventions

코드, 문서, workflow 규약을 보관한다.

상위 `conventions/`에는 harness-helm 용어, 개념, 반복 규약처럼 넓게 참조되는 기준 문서를 둔다.

하위 `conventions/guidelines/`에는 h2 workflow에서 agent나 사용자가 따라야 하는 구체적 운용 지침을 둔다.

## 파일 패턴

```text
docs/40_knowledge/conventions/{topic}.md
docs/40_knowledge/conventions/guidelines/{topic}.md
```

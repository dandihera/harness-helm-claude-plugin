# 40_knowledge

검증된 해결책, 규약, 학습 내용, 외부 비교 참고자료를 보관한다.

## 하위 분류

- `solutions/`: 재사용 가능한 문제 해결 패턴
- `conventions/`: 코드, 문서, 운영 규약
- `learnings/`: 검증된 학습과 회고에서 얻은 지식
- `references/`: 외부 repository, tool, skill ecosystem을 compact하게 정리한 비교 참고자료

## 사용 기준

- 검증 전 초안은 `.harness-helm/runs/**/promotion-candidates/`나 01~04 산출물에 남긴다.
- canonical 지식으로 승격하려면 owner와 검증 근거가 있어야 한다.
- `references/` 문서는 harness-helm의 source of truth가 아니다. 외부 자료를 다시 확인해야 하는 판단은 `source_references`를 따라 원문을 재검증한다.

# 10_domain

도메인 용어, 업무 규칙, 조직 공통 개념을 보관한다.

## 사용 기준

- 반복해서 참조되는 업무 용어와 규칙만 둔다.
- 기능 작업 중 나온 초안은 검증 전까지 `routing.promotion_candidates`로 남긴다.
- 검증된 항목은 `domain` template으로 작성한다.

## 파일 패턴

```text
docs/10_domain/{domain}/{topic}.md
```
